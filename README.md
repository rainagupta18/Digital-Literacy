# Digital-Literacy
Digital literacy is an essential skill in today’s world where students rely heavily on technology for learning, communication, and collaboration. This project was undertaken as part of the Digital Literacy course to understand how digital tools can be used effectively, responsibly, and professionally.
<br>
As a Student Digital Ambassador, the goal was to explore various aspects such as online presence, communication etiquette, cybersecurity awareness, and collaboration tools. Through this project, I learned not only how to use digital platforms but also how to behave responsibly in a digital environment.
